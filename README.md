# README

## Lint
```
  bundle exec rubocop -A    
```


* Ruby version
```
ruby 2.7.5
```

* System dependencies
```
 bundle install
```

* Configuration

* Database creation
```
   rails db:create
```

* Database initialization
```
   rails db:migrate
```

* Seed database
```
   rails db:seed
```

* How to run the test suite
```
  rails test
```

* How to run the project
```
 rails server
```

* Services (job queues, cache servers, search engines, etc.)

* Apply schedules
```
  whenever --update-crontab
```

* Run Sidekiq
```
  bundle exec sidekiq
```

* Generate Doc
```
rake swagger:docs
```
